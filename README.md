## パラメータ
### doPost引数
LINEの標準入力に準拠
contents.events[0]以下は以下の通り。

|リクエストボディ|必須|概要|
|---|---|---|
|.message.text|必須|テキスト|
|.replyToken|必須|トークで使用しているが、グループからも送信されるので必須とする|
|.source.groupId||LINEグループID|

### 戻り値
JSON形式

|パラメータ|出力例|
|---|---|
|result|True/False|
|message|リクエストボディの`.message.text`と同じ|
|groupId|リクエストボディの`.source.groupId`と同じ|
